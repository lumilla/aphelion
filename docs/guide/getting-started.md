# Getting Started

## Installation

Install via npm:

```bash
npm install @lumilla/aphelion
```

## Peer Dependencies

For React use we recommend React 19+. React is optional; Aphelion works fine with vanilla JavaScript.

## Importing Styles

Don't forget to import the stylesheet:

```javascript
import "@lumilla/aphelion/styles.css";
```

Or in your CSS/SCSS:

```css
@import "@lumilla/aphelion/styles.css";
```
